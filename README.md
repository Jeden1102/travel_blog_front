# Travel Blog front

Simple travel blog with Strapi DB as a headless CMS.

# Technologies used

- Nuxt3
- Taiwlind
- Typescript
- Google maps API
- AOS

## Installation

1. Build and run dev.

```bash
npm run install
npm run dev
```
2. Copy env.example to .env and insert keys.

```bash
cp .env.example .env
```
